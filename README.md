# 🎬 Filmes API — CRUD Básico com Node.js

Este projeto é o meu marco zero no desenvolvimento Backend, seguindo as aulas do Professor Edson Maia. Nele, implementei uma API REST para gerenciar uma lista de filmes, focando nos fundamentos de roteamento e verbos HTTP.

## 🚀 Tecnologias
- Node.js
- Express
- Nodemon (Desenvolvimento)
- Thunder Client (Testes de API)

## 🧠 O que eu aprendi:
- **Estrutura de API**: Configuração de servidores com Express e uso de Middlewares (`express.json`).
- **CRUD (Read & Create)**: 
  - Listagem completa e busca filtrada por ID usando `req.params`.
  - Cadastro de novos itens via `req.body` (Método POST).
- **Status Codes**: Tratamento de erros com `404 Not Found` e confirmação de criação com `201 Created`.
- **Developer Experience**: Automação de scripts no `package.json`.

## 🛠️ Como rodar o projeto
1. Clone o repositório.
2. Instale as dependências: `npm install`.
3. Inicie o servidor: `npm run dev`.
4. Acesse `http://localhost:3000/filmes`.
