# 📓 Diário de Bordo — Gabriel (Node.js & Express)

## 📅 16/05/2026 — O Marco Zero do Backend

### 🚀 O que aprendi hoje:
1. **Setup de API**: Como iniciar um projeto com `npm init -y` e configurar o `package.json`.
2. **Nodemon**: Ferramenta para não precisar reiniciar o servidor manualmente (`npm run dev`).
3. **Rotas com Express**:
   - `app.get()`: Para ler dados (ex: listar filmes ou buscar por ID).
   - `app.post()`: Para registrar novos dados.
4. **O Tradutor (Middleware)**: A importância do `app.use(express.json())` para capturar o `req.body`.
5. **Status Codes**: 
   - `200`: Sucesso!
   - `201`: Criado com sucesso (usado no POST).
   - `404`: Não encontrado.
